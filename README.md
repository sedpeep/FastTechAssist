# FastTechAssist

FastTechAssist is a mobile application designed for seamless issue reporting and resolution in technical and administrative environments. It features dedicated interfaces for Faculty, Technical Staff, and Admin users, ensuring efficient workflows tailored to their needs.

---

## Features

### Faculty Side:
- **Report an Issue:** Faculty can report issues with detailed descriptions and track their statuses.
- **View Tickets:** Faculty can view their ongoing and resolved tickets.
- **Edit Profile:** Update user profile details like name, email, and phone.
- **Terms & Conditions:** Access app usage policies and terms.
- **Logout and Account Deletion:** Faculty can log out or delete their accounts.

### Technical Staff Side:
- **Available Tickets:** View and claim tickets.
- **Processing Tickets:** Manage and resolve tickets in progress.
- **Completed Tickets:** View resolved issues.
- **Settings:** Update profile and set availability status.

### Admin Side:
- **Resolved Issues:** View all resolved reports.
- **Open Issues:** Search and filter issues by report numbers, description, or type.
- **Settings:** Manage admin account with profile editing and logout options.
